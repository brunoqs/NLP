quarta-feira, 12 de dezembro de 2018Muitos que usam notebooks tem uma “guerra infinita” (sério, sem trocadilhos =) ) ao tentar economizar a maior quantidade de bateria possível, assim podendo utilizar o notebook por mais tempo. A ferramenta que utilizaremos se chama TLP.O TLP é ferramenta via linha de texto que permite que você controle módulos do kernel e ajuste o consumo de dispositivos de forma individual e bem detalhada, entretanto, como todo aplicativo que você for usar via terminal, é necessário um certo conhecimento técnico, o que pode ser drasticamente reduzido com a sua interface.Uma interface para o TLPNo Windows é menos comum de acontecer, mas no mundo Linux o software e a interface para ele não são necessariamente a mesma coisa, inclusive, é possível usar, em alguns casos, várias interfaces diferentes para o mesmo software.E o TLPUI é uma interface gráfica para o TLP e permite que você gerencie o TLP de uma forma mais simples e fácil. Fizemos um artigo bem completinho explicando como funciona o TLP.Tanto o TLP, quanto a sua interface são distribuídos para Ubuntu e Linux Mint através de um repositório PPA, e como você deve saber, é possível adicionar PPAs sem usar o terminal, entretanto, para fins de versatilidade nós vamos usá-lo aqui neste tutorial.Instalando o TLPPrimeiramente vamos instalar o TLP para depois instalar o TLPUI. Vamos abrir o terminal e digitar os seguintes comandos:Depois de instalado é muito simples achar o TLPUI, basta acessar o menu e digitar “tlpui” e vai aparecer a seguinte janela do programa.Essa facilidade é muito bem vinda, ainda mais para quem não tem familiaridade com o terminal e queira fazer a gestão de energia, o TLPUI vem para ajudar e muito essas pessoas, visto que a mudança pode ser feita a um click ou dois, facilitando a vida delas.Como usar?Na primeira tela podemos ter uma visão geral do que o TLP faz, como habilitar e desabilitar, qual modo padrão queremos ( bateria ou ac, que seria o plugado na tomada).Outra aba interessante é a “Disk”, onde podemos configurar melhor como o nosso HD vai se comportar em cada situação (via bateria ou não). Lembre-se de tomar cuidado ao habilitar/desabilitar cada opção, faça isso com muita cautela, pois isso vai afetar a forma com que a sua máquina vai funcionar em algumas circunstâncias.A aba “Network”, podemos configurar como o Wi-Fi vai se comportar, visto que o Wi-Fi ligado junto com o bluetooth (quando disponível) é um dos maiores consumidores de energia quando não utilizados, juntamente com um grande brilho da tela. Então nessa aba podemos configurar isso.Mais uma aba interessante é a que se pode configurar o “USB”, em que podemos configurar como as portas USB vão se comportar quando temos um dispositivo conectado, como um Smartphone, ou quando se tem uma impressora plugada e afinsPara saber mais de cada função do TLP, acesse o site deles, lá você encontra uma documentação bem detalhada.
