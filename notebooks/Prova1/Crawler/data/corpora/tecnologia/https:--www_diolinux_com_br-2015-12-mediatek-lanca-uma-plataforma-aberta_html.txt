MediaTek lança uma plataforma aberta para internet das coisasExistem muitas empresas que estão tentando puxar o setor de desenvolvimento da chamada "Internet das Coisas", entretanto, apesar de muitas vezes os softwares utilizados serem de código aberto o a plataforma de onde eles são desenvolvidos não é, pelo menos até a MediaTek dar início a este projeto.quarta-feira, 9 de dezembro de 2015Existem muitas empresas que estão tentando puxar o setor de desenvolvimento da chamada "Internet das Coisas", entretanto, apesar de muitas vezes os softwares utilizados serem de código aberto o a plataforma de onde eles são desenvolvidos não é, pelo menos até a MediaTek dar início a este projeto.Quem pretende desenvolver para a "Internet das Coisas" agora tem um novo aliado, a MediaTek acabou de anunciar uma nova plataforma para desenvolvimento deste tipo de "device", a plataforma chama-se LinkIT Smart 7688 e é ideal para trabalhar com dispositivos avançados baseados em conexões Wi-Fi. De acordo com a MediaTek a plataforma foi pensada para trabalhar com "câmeras IP", dispositivos de vigilância, aparelhos inteligentes e gateways de Wi-Fi que fazem o uso de serviços cloud.
