segunda-feira, 17 de dezembro de 2012Mais uma notícia que vem provar que a Microsoft deu um tiro na água com o Windows 8Windows sem aplicativos?Isso pode até soar estranho ( porque é de fato ) mas uma notícia que li num site britânico falava a respeito das pretensões do Google com o Windows 8, tanto em PCs como em dispositivos móveis.Segundo o site o Google não estaria interessado em desperdiçar tempo e dinheiro aprimorando e adaptando aplicativos, como o Gmail e Gtalk, para plataformas que rodem o Windows 8.Motivo? Simples, "Vamos investir onde a maioria dos usuários está" disse Clay Bavor, Diretor de Gerenciamento de produto do Google Apps.Claro, o Google não deixaria seus usuários sem nenhum aplicativo, mas Clay deixou claro que esta não é realmente a prioridade, o Windows 8 causa tanta controvérsia que a Microsoft já anda pensando em lançar uma nova versão, que na verdade seria uma atualização para o W8, no próximo ano, possivelmente chamando-a de Windows Blue.Clay ainda deixou claro que o Google é muito sensato em onde investir seu dinheiro, "agora não é a hora, mas se a coisa mudar... certamente investiremos", disse.Opinião do AutorNa verdade até não tem muito que opinar, para mim o efeito Windows 8 é claro, ele não foi aceito com bons olhos porque mexeu demais com a maneira com que os usuários utilizam os computadores, e não é a primeira vez que vemos a Microsoft tentando empurrar algum software "na marra" para as pessoas.O problema da Microsoft e do Windows 8 são os próprios usuários que não se convenceram o suficiente a ponto de largar o Windows 7 ou mesmo o XP.Se os usuários não querem o Windows 8, porque os fabricantes de Hardware vão embarcar o Windows 8 nas suas plataformas? Por que grandes empresas como o Google investiriam tempo e dinheiro para desenvolverem apps para entrarem no Market do Windows?Particularmente e para quem usa Linux a algum tempo, sabe que ambientes como o Gnome-Shell, Unity e o KDE trazem uma integração com a internet que o Windows tentou implementar somente nessa versão, o Windows 8, além da interface Metro não traz muitas novidades para nós. A única coisa realmente positiva que consegui encontrar foi o boot realmente muito rápido, no mais ainda é o mesmo Windows de sempre.Quer continuar por dentro de tudo o que acontece por aqui?Então curta nossa página no FACEBOOK, siga o blog no TWITTER - Siga também nossos escritores
