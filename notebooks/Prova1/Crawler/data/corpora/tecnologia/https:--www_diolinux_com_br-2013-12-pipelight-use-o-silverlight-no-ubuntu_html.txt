sexta-feira, 20 de dezembro de 2013Como usar o Plugin Silverlight no Ubuntu com o PipelightEste é o problema de empresas que fazem programas de código fechado estabelecerem um padrão, por conta disso todos ficam a mercê de suas vontades, um desses casos é o Silverlight, um plugin Microsoft que permite executar conteúdo multimídia em navegadores de internet.A Microsoft não tem porque se preocupar em portar o Silverlight para Linux por isso criou-se uma alternativa open source para o mesmo, o projeto chama-se Pipelight e ainda está em desenvolvimento mas já tem um bom suporte e um bom funcionamento.Para instalar no Ubuntu vamos precisar usar um PPA:sudo add-apt-repository ppa:pipelight/stablesudo apt-get updatesudo apt-get install pipelightCom este PPA você sempre terá a versão mais atualizada do Pipelight e poderá acessar rádios online que usem o Silverlight por exemplo.
