7 Softwares poderosíssimos que são open sourceAlguém ainda duvida que o modo open source de desenvolvido é o futuro? Na verdade, para muitos ele é o presente também, hoje vamos mostrar para você 7 softwares incríveis que são de código aberto, algo que na cabeça de muitos é algo "estranho".quarta-feira, 9 de dezembro de 2015Alguém ainda duvida que o modo open source de desenvolvido é o futuro? Na verdade, para muitos ele é o presente também, hoje vamos mostrar para você 7 softwares incríveis que são de código aberto, algo que na cabeça de muitos é algo "estranho"."Se fosse bom teria o código fechado"É incrível como essa frase acima ainda é viva em algumas mentes, ter código fechado ou ser pago simplesmente não com que o programa seja bom, não necessariamente, é importante quebrar o preconceito que muitas vezes existe com softwares que tem código aberto, porque podemos ver que existe uma crescente muito grande neste modelo de desenvolvimento, grandes empresas o utilizam e você pode também.Separamos aqui 7 softwares muito poderosos, utilizados por empresas de tecnologia de ponta e que tem código aberto.1 - TensorFlowO TensorFlow é o projeto open source desenvolvido pela Google para criar uma máquina de buscas com inteligência artificial, falamos mais dele em uma outra matéria no Diolinux.2 - SpinnakerA Netflix o Spinnaker para fazer as atualizações de software por toda a grande infraestrutura da empresa, recentemente eles lançaram o código no projeto no GitHub, ele funciona não somente com a plataforma Netflix, mas também com a AWS (Amazon), com o Google Cloud e futuramente trabalhará também com o Microsoft Azure.3 - FluxEm 2014 o Facebook abriu o código do Flux, a aplicação web que a empresa utiliza para criar interfaces de aplicações para clientes, tida a importância da rede social a massividade de dados que a empresas trabalha, um software desta importância ter código aberto para ele seja aprimorada constantemente faz muito sentido.4 - KubernetesA aplicação por detrás do sistema de gestão de clusters Borg do Google, a estrutura do contentor Kubernetes de código aberto torna-a um importante suporte do ecossistema de servidores da empresa. Se você quiser conhecer melhor o Kubernetes eu recomendo um vídeo muito legal do canal Linux Tips, o vídeo é em português apesar do nome do canal. ;)5 - SwiftEste é um dos mais recente a entrar para esta lista, o Swift teve seu código fonte aberto pela Apple nesta semana, o Swift é a linguagem de programação utilizada para criar Apps para o ecossistema da "maçã", ou seja, até a empresa de Cupertino, reconhecida por manter suas tecnologias sempre trancadas agora está mudando um pouco de opinião.6 - GrommetO Grommet da HP é uma estrutura de design de interfaces também disponível com código aberto, basicamente o Grommet trabalha como um conjunto de ferramentas para ajudar os desenvolvedores a interfaces gráficias uniformes, amigáveis e úteis.7 - PrestoEsta é outra ferramenta de código aberto desenvolvida pelo Facebook, é o Presto que faz a "mágica" de analisar os petabytes de dados que a rede social gera, é ele que consegue analisar todas as informações dos usuários fazendo dele uma peça chave da publicidade no Facebook.
