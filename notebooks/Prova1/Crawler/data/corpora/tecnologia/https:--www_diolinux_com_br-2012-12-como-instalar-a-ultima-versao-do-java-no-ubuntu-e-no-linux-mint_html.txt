quinta-feira, 6 de dezembro de 2012Tutorial de instalação do Java no Ubuntu e no Linux MintTudo bem pessoal?
Hoje vou trazer uma dica muito legal, sabemos que o OpenJDK ,"o Java OpenSource", nem sempre funciona em algumas situações, sites bancários e alguns aplicativos que rodam somente sobre o Java oficial feito pela Oracle são bons exemplos.Para instalar o Java 8 no seu Ubuntu ou no Linux Mint ( e derivados de ambos) vamos nos utilizar de um PPA produzido pela equipe do webupd8team.InstalaçãoAbra o terminal e cole os seguintes comandos sequencialmente, apertando a tecla "enter" após cada um, na primeira linha que você colar será pedido a senha de ROOT, digite-a, ela não vai aparecer, mas não se preocupe é assim mesmo.sudo add-apt-repository ppa:webupd8team/javasudo apt-get updatesudo apt-get install oracle-java8-installerPronto pessoal, Java 8 Instalado, para ter certeza se tudo ocorreu direitinho, ainda no terminal cole o seguinte comando:java -versionEste comando deve apresentar a versão do Java instalado no computador.Quer continuar por dentro de tudo o que acontece por aqui?Então curta nossa página no FACEBOOK, siga o blog no TWITTER - Siga também nossos escritores
