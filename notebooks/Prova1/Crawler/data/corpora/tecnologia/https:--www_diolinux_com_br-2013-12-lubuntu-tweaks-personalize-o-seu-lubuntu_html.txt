sexta-feira, 20 de dezembro de 2013Lançado Software para configuração do LubuntuVocê que usa o Ubuntu a um tempinho já deve ter ouvido falar na aplicação Ubuntu Tweak que é uma ferramenta para fazer uma série de configurações no sistema.Agora se você usa o Lubuntu ( Ubuntu+LXDE) também tem uma ferramenta para fazer ajustes no sistema, este é o Lubuntu Tweaks, essa ferramenta ainda é recente e não tem muitos recursos mas tende a evoluir e aumentar as configurações com o tempo.
Atualmente o Lubuntu Tweaks permite que você configure por exemplo os ícones na área de trabalho, para instalar o Lubuntu Tweaks no Lubuntu adicione o seguinte PPA:sudo add-apt-repository ppa:lubuntu-dev/non-official-appssudo apt-get updatesudo apt-get install lubuntu-tweaksAté a próxima dica!
