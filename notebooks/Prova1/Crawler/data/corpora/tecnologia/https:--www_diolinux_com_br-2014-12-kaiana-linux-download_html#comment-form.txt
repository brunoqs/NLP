Lançado beta do Kaiana Linux, uma nova distro brasileiraterça-feira, 23 de dezembro de 2014Depois de muito tempo desenvolvendo o sistema o projeto União Livre anunciou que o Kaiana, a distro Linux mantida por eles, está em fase Beta final, sendo liberada para o público interessado para que possam ser reportados os últimos bugs no sistema.Baixe agora o Kaiana LinuxEste é um projeto muito bem organizado entre pessoas que tem o interesse de desenvolver software no Brasil, o sistema tem o conceito parecido com o Big Linux, uma distro Linux bem famosa a alguns anos atrás, talvez porque o fundador do Big Linux, Bruno Gonçalves, também faça parte deste projeto.Segundo as notas de lançamento, que você pode ler aqui, o sistema usa como base o sólido Ubuntu 14.04 LTS, o que faz com que o mesmo tenha suporte até 2019, usando ambiente gráfico KDE 4.12, com uma série de aplicações que visam facilitar o uso do computador no dia a dia.Apoio ao KaianaO Diolinux firmou uma parceria com a União Livre e agora na DioStore contamos com ítens, camisetas e canecas, com a logo do Kaiana Linux, uma porcentagem do lucro das vendas destes produtos serão revertidas ao projeto Kaiana para que o desenvolvimento de software livre no Brasil continue progredindo.
