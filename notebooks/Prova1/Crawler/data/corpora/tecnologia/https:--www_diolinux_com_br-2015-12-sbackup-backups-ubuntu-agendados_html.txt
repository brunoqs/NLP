sexta-feira, 4 de dezembro de 2015Hoje você vai conhecer o Sbackup, ou Simple Backup, uma ferramenta muito interessante para você gerenciar Backups no seu computador e até mesmo agendar Backups automáticos, conheça.Faça Backups facilmente com o SBackupManter os seus arquivos à salvo é algo muito importante, acho que não é necessário nem falar mais hoje em dia o quanto um bom Backup é útil, então que tal automatizar esta tarefa? O SBackup é uma aplicação bastante interessante e que vai te ajudar nisso.FuncionalidadesCom o Sbackup você pode escolher que pasta o programa fará Backup, incluir arquivos no Backup ou evitar que alguma pasta seja inclusa no Backup, ele ainda compacta esse Backup em formatos Gzip, Bzip2, você pode agendar os Backups para serem feitos semanalmente, diariamente, ou especificar o intervalo dos dias, selecionar onde os arquivos de Backup serão guardados e mais algumas coisas.Como instalarSe você usa uma distribuição Linux diferente do Ubuntu ou Linux Mint procure no repositório de softtware do sistema pelo pacote "sbackup" sem as aspas, se você usa o Ubuntu ou algum derivado como o Linux Mint, basta clicar no botão abaixo:Opcional: Se você prefere fazer as coisas pelo terminal, o comando para instalar o programa é este:sudo apt-get install sbackupDepois de instalado o programa, ele estará no menu do seu sistema através de dois ícone, pesquise por Sbackup e você verá, um deles serve para configurar o programa (que é esta imagem que ilustra o post), o outro serve para você restaurar o Backup feito anteriormente (veja a imagem abaixo), você pode usar ele de uma forma parecida aos pontos de restauração do Windows.Dica adicionalNão que você precise fazer isso mas achei a medida interessante, se você usa algum serviço de armazenamento em nuvem como o Dropbox, Copy, Mega, etc, você pode direcionar a pasta de Backup do Sbackup para dentro de uma das pastas de sincronia destes aplicativos, assim, além de fazer backup local você também faz backup em nuvem.Se você achou interessante o Sbackup mas não se convenceu, além do DejàDup que já vem com o Ubuntu por padrão, outra opção interessante é o SystemBack, ele tem um recursos bastante interessante para criação de liveCDs do sistema completo, vale a pena conhecer.Até a próxima!
