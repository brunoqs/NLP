Extenções para Gnome-shellsexta-feira, 30 de dezembro de 2011Esta dica é para você que gostou mais da cara do gnome 3 do que a do Unity ( se for usuário Ubuntu pelo menos), mas acredita que faltam alguns detalhes para deixar o G3 perfeito.Como todo novo projeto em desenvolvimento, coisa que é bem típica do mundo Open Source, a tendência é que facilidades por assim dizer sejam implementadas com o tempo.Agora vou mostrar como deixar o Gnome 3 mais ágil instalando extenções do Shell, como atalhos de locais e até uma dock, então "demo via let's go!":O vídeo acima serve de ilustração para o que pretendemos fazer.
Bom, antes de instalar as extensões propriamente ditas, vamos instalar uma ferramenta para gerencia-las.
Uma ferramenta chamada, gnome-tweak-tool, que é necessária para que depois possamos configurar essas extensões.
Ela pode ser instalada facilmente através da central de programas do Ubuntu, com o nome de "Configurações Avançadas", mas você pode simplesmente digitar Gnome-tweak-tool na central e instalar o pacote, ou ainda pelo terminal:sudo apt-get install gnome-tweak-toolAlternative-status-menu: serve para aqueles que querem um botão visível o tempo todo, o menu de estado substitui o GNOME Shell com uma apresentação em separado de suspensão e desligamento. Ele também adiciona a opção de Hibernate (sudo apt-get install gnome-shell-extensions-alternativo-status-menu).Auto-move-windows: permite que você gerencie mais facilmente espaços de trabalho, dando um espaço de trabalho específico para cada aplicação, logo que abrir uma janela, além da capacidade de definir uma senha GSettings (sudo apt-get install gnome-shell-extensions-auto-mover-windows).Dock: mostra um integrador de tarefas do lado direito da tela (sudo apt-get install gnome-shell-extensions-dock).Xrandr-indicator: indicador substitui gnome-settings-daemon baseado em GTK + para um nativo. Permite ao usuário rodar o monitor do laptop e abrir a tela de preferências rapidamente (sudo apt-get install gnome-shell-extensions-xrandr-indicador).Native-window-placement: algoritmo alternativo para o layout das miniaturas na janela de visualização, que reflete com mais precisão as posições e tamanhos reais (sudo apt-get install gnome-shell-extensions-native-window-colocação).Places-menu: exibe um indicador de status para a navegação em alguns lugares (sudo apt-get install gnome-shell-extensions-lugares-menu).Workspace-indicator: é um menu que indica o espaço de trabalho atual e permite mudanças rápidas. Usa os nomes dos espaços de trabalho configurado no GConf (sudo apt-get install gnome-shell-extensions-workspace-indicador).Uma outra maneira de instalarmos as extensões é usando o synaptic, como ele não vem mais instalado por padrão no Ubuntu vamos precisar instala-loNo terminal digite:sudo apt-get install synapticDepois disso, você pode chamar o synaptic pelo terminal usando o comando " sudo synaptic" (sem aspas") e digitar no campo de buscas, "shell extencions", e instalar as que desejar.Para ativar o desativar as extensões, basta abrir o gnome-tweak-tool pelo menu mesmo e na abra de extensões, clicar na opção ON daquelas que você quer que fique ativas.
