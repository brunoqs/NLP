sexta-feira, 5 de dezembro de 2014Agora você vai poder usar esta interessante funcionalidade no Facebook no seu Linux também, agora o chat por vídeo funciona também no sistema operacional livre, saiba mais:Chat do Facebook em vídeo agora no UbuntuÉ isso mesmo, boas novas para quem sempre quis utilizar o chat por vídeo do Facebook no Linux, agora a funcionalidade tem suporte ao Pinguim.Depois de uma atualização do sistema percebi este pequeno detalhe, porém não sei dizer se é culpa da atualização das APIs que estavam listadas no gerenciador de atualizações ou atualização do Firefox que recebeu suporte a WebRTC.De qualquer forma percebi que o iconezinho da câmera que aparece no chat do Facebook por enquanto aparece apenas no Firefox, no Google Chrome é possível receber chamada alheias mas não aparece o ícone para iniciar uma chamada.Acredito que este problema deverá ser solucionado e o chat do Facebook deverá funcionar também no Chrome uma vez que o Opera, que recebeu uma nova versão recentemente, é baseado no Chromium, mesma base no Chrome é do Facebook.
