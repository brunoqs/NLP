GRID Autosport chegará ao Linux ainda este ano!Mais um game de peso está chegando ao sistema do pinguim, um jogo que vai saciar um pouco da sede dos jogadores Linux por um bom game de corrida, o game deverá chegar ainda este mês na Steam para a plataforma aberta.sexta-feira, 4 de dezembro de 2015Mais um game de peso está chegando ao sistema do pinguim, um jogo que vai saciar um pouco da sede dos jogadores Linux por um bom game de corrida, o game deverá chegar ainda este mês na Steam para a plataforma aberta.Um ótimo game de corrida chegando ao LinuxPara quem estava sentindo falta, agora além do excelente Dirt Showdown, teremos também GRID Autosport para Linux, a previsão é que o game saia no dia 10 de Dezembro.O game foi lançado ainda em 2014 para Windows, PlayStation e XBox e agora poderemos jogá-lo no Linux também.Além de belos carros e pistas, GRID é conhecido por ser um ótimo simulador de pilotagem, um pouco diferente dos títulos Arcades que temos no momento, mas sem dúvida será uma ótima opção para o seu divertimento, ainda mais se você gostar deste tipo de game.A empresa que vai fazer o porte do game para Linux, a Feral Interactive, também fará o porte do mesmo para Mac OSX, esta é a principal empresa de porte de games atualmente, recentemente eles fizeram também o porte de Alien: Isolation para o Linux.
