terça-feira, 8 de dezembro de 2015Os desenvolvedores do elementary OS, uma das mais populares distribuições Linux para desktop, migraram do Launchpad para o GitHub, ou pelo menos os seus ícones.Cassidy James, um dos principais desenvolvedores da distribuição anunciou hoje que agora os ícones do elementary OS estarão disponíveis no GitHub e não mais no Launchpad, onde ainda se encontra toda a base de software do elementary.A mudança parece ser por gosto pessoal mas vem fundamentada no argumento de permitir forks mais facilmente, ele comenta que no início deste ano o elementary abriu o código de seu site no Git e muitas pessoas fizeram cópias do projeto, ele espera que o mesmo possa acontecer com os ícones.Os ícones, assim como o restante do design, são sem dúvidas o fator de maior destaque do elementary, o Pantheon é uma ambiente gráfico bonito e talvez no Git o projeto tenha mais visibilidade por parte de outras pessoas que não fazem necessariamente parte do mundo Linux, ao contrário do que acontece no Launchpad que é focado em armazenar projetos e softwares para toda a família Ubuntu.
